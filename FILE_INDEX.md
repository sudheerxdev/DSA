
3. Snippet appears in full
4. Edit highlighted parts

## 🚀 YOUR FIRST CODING SESSION

### Step-by-Step:

1. **Open VS Code**
   - Point to `d:\DSA` folder

2. **Read START_HERE.cpp**
   - See all quick info

3. **Create new file**
   - `Ctrl+N` → `solution.cpp`

4. **Get template**
   - Type: `dsa`
   - Press: `Tab`

5. **Write your code**
   - Replace `// Your code here` section

6. **Build & Run**
   - Press: `Ctrl+Alt+R`
   - See output in terminal

7. **Debug if needed**
   - Press: `F5`
   - Set breakpoints with: `F9`
   - Step through code with: `F10` / `F11`
